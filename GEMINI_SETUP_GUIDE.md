# Guide: Setting Up Your Marigold AI Assistant
Welcome! Marigold AI Assistant uses artificial intelligence (AI) to help you search civic data using plain, everyday language. Because AI requires computing power, we ask each volunteer or agency employee to set up their own **API Key** (think of it like a personal passcode) so that Marigold can connect to Google's AI model.

This guide will walk you through getting your API key in about 5 minutes. No technical experience required!

## 💡 What You Need to Know: Free vs. Paid (Pro)
Google offers two tiers for their Gemini AI:
1. **Free Tier (Recommended):** This is completely free! It is perfect for standard civic validation work. The only drawback is that during global peak hours, it might occasionally run a little slower.
2. **Pay-as-you-go (Pro):** If your jurisdiction is running massive batch audits and needs guaranteed priority speeds, you can attach a payment method. However, **we highly recommend starting with the Free Tier** as it is more than enough for everyday use.

---

## 🛠️ Step-by-Step Instructions

### Step 1: Sign in to Google
You will need a Google account (like a Gmail address or Google Workspace agency account).
1. Go to [Google AI Studio](https://aistudio.google.com/app/apikey).
2. If you aren't already signed in, Google will ask you to log in.

### Step 2: Accept the Terms
1. When you first log in to AI Studio, you may see a welcome screen with Terms of Service.
2. Read the terms, check the boxes to agree, and click **Continue**.

### Step 3: Create Your API Key
1. On the left-hand side of your screen, click the button that says **"Get API key"**.
2. Look for the large blue button that says **"Create API key"** and click it.
3. A small window will pop up asking you to select a project. Just click the button that says **"Create API key in a new project"**.
4. Google will generate a long string of letters and numbers. This is your API key! **Keep it secure**, just like a password.

### Step 4: Copy and Paste
1. Click the **"Copy"** button next to your new API key.
2. Open the **Marigold Insights** application.
3. Navigate to **Group Settings** in the left sidebar.
4. Look for the section labeled **"API Configuration"** and paste your key into the **Gemini API Key** box.
5. Click **Save Settings**.

---

## 🎉 You're All Set!
That's it! Your Marigold AI Assistant is now active. You can navigate to the **AI Assistant** module and start typing natural language queries.

*If you ever need to rotate or replace your key, you can always log back into [Google AI Studio](https://aistudio.google.com/app/apikey), delete the old key, and create a new one.*
