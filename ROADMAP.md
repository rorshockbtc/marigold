# Marigold Insights: Product Roadmap & Strategic Vision 🌻

Marigold Insights is dedicated to modernizing civic data collaboration, voter roll verification, and non-partisan statistical auditing through an offline-first, zero-cloud client architecture. This roadmap outlines our strategic milestones across product engineering, pilot deployments, and institutional scaling.

---

## 🟢 Phase 1: Alpha Pilot Deployment (Current Horizon - Q3 2026)
**Focus:** Proving zero-cloud computational reliability and statistical precision in real-world civic environments.

* [x] **Client-Side Web Worker Engine:** Stream processing of CSV files up to 2GB in-memory without browser freezes or cloud database uploads.
* [x] **Probabilistic Fellegi-Sunter Linkage:** Log-odds scoring engine enriched with Jaro-Winkler prefix weighting and phonetic nickname dictionaries (`fellegi-sunter.ts`).
* [x] **Canonical 1,000-Pair Benchmark Suite:** Open-source statistical verification harness achieving 100% Precision and Recall across typographical noise and NCOA relocations (`benchmark-suite.ts`).
* [x] **Mississippi Fair Elections Pilot:** Initial deployment and alpha testing with real-world state volunteer coordinators using synthetic privacy-protected datasets.
* [ ] **Pilot Feedback Loop:** Incorporate direct feedback from pilot users to refine anomaly explanation badges and CSV export workflows.

---

## 🟡 Phase 2: National Cartridge Store & Ecosystem (Q4 2026)
**Focus:** Expanding the decentralized marketplace of standardized civic audit playbooks.

* [ ] **Cryptographic Cartridge Registry:** Implement automated SHA-256 signing pipelines for community-submitted JSON audit playbooks.
* [ ] **Multi-State NCOA Templates:** Standardized relocation checklists tailored for specific state statutory requirements (e.g., Ohio, Florida, Texas, Georgia).
* [ ] **Assessor Zoning Cross-Reference Module:** Interactive UI tools for mapping commercial property tax roll land-use codes against residential registration files.
* [ ] **Offline PWA Support:** Full Progressive Web App (PWA) offline caching so county auditors can execute field checks without active internet connections.

---

## 🔵 Phase 3: Institutional Scaling & Grant Alignment (Q1 2027)
**Focus:** Transitioning from alpha pilots to funded municipal deployments and foundation grants.

* [ ] **Philanthropic Grant Submissions:** Alignment with Knight Foundation, Google.org, and civic tech innovation funds using our published benchmarks and security whitepapers.
* [ ] **State Administering Agency (SAA) Partnerships:** Formal B2B software license agreements with county election commissioners and state IT directors.
* [ ] **Enterprise Technology E&O Insurance:** Execution of full Errors & Omissions coverage tailored for public sector civic data deployments.

---

## 🟣 Phase 4: Formal SOC 2 Type II Certification & SDKs (Q2 2027+)
**Focus:** Enterprise maturity and developer ecosystem expansion.

* [ ] **Formal CPA SOC 2 Audit:** Transitioning our bootstrapping manual audit ledger (`audit-ledger.md`) into a certified Type II continuous compliance report.
* [ ] **Marigold Autonomous Agent SDK:** Integration with Google Antigravity and multi-agent AI frameworks for automated anomaly detection and natural language data querying.
* [ ] **Cross-Jurisdictional Federation:** Encrypted zero-knowledge proof (ZKP) protocols allowing neighboring counties to verify cross-border relocations without sharing raw citizen PII.
